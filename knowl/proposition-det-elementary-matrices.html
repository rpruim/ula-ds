Proposition 4.5.13. The determinants of elementary matrices.
  • If \(S\) is a scaling matrix with scaling factor \(k\text{,}\) then \(\det(S) = k\text{.}\)
  • If \(P\) is an interchange matrix, then \(\det(P) = -1\text{.}\)
  • If \(R\) is a replacement matrix, then \(\det(R) = 1\text{.}\)
