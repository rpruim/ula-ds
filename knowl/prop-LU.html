Proposition 4.7.2. \(PLU\) factorization: \(LU\) factorization with partial pivoting.

For any \(m \by n\) matrix \(A\) there are matrices \(P\text{,}\) \(L\text{,}\) and \(U\) such that
  • \(P\) is an \(m \by m\) (row) permutation matrix.
  • \(L\) is an \(m \by m\) lower triangular matrix.
  • \(U\) is an \(n \by n\) uppter triangular matrix.
  • \(PA = LU \text{.}\)
Matrices \(P\text{,}\) \(L\text{,}\) and \(U\) can by found by keeping track of the elementary matrices corresponding to Gaussian elimination with row permutations to put numbers with the largest absolute value into the pivot positions.
We can solve \(A \xvec = \bvec\) using this factorization as follows:
  • Rewrite as \(PA\xvec = LU \xvec = P\bvec\text{.}\)
  • Solve \(L \cvec = P \bvec\) for \(\cvec\) using backsubstitution.
  • Solve \(U \xvec = \cvec\) for \(\xvec\) backsubstitution.
  • This implies that \(P\bvec = L\cvec = LU \xvec = PA \xvec \text{,}\) so \(A\xvec = \bvec\) since \(P\) is invertible.
