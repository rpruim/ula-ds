We may ask Python to create the \(2\by4\) matrix
\begin{equation*} \left[ \begin{array}{rrrr} -1 \amp 0 \amp 2 \amp 7 \\ 2 \amp 1 \amp -3 \amp -1 \\ \end{array} \right] \end{equation*}
by using nested lists (a list of lists). But just as we did with vectors, we will convert this to a numpy n-dimensional array using np.array(). The shape of the resulting array tells us the number of rows and columns in the matrix.
