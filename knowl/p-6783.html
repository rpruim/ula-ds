There are two versions of \(QR\) factorization. The version introduced here is sometimes called the reduced \(QR\) factorization. When \(A\) is \(m \by n\) and \(m > n\text{,}\) the full \(QR\) factorization can be created by augmenting \(Q\) to a square (\(m \by m\)) orthogonal matrix \(\left[ Q Q+ \right]\text{.}\) Similarly, we can add 0â€™s below \(R\) to create an \(m \by n\) matrix. When we multiply these expanded matrices we get
\begin{equation*} \begin{bmatrix} Q \amp Q+ \end{bmatrix} \begin{bmatrix} R \\ 0\end{bmatrix} = A \end{equation*}
because all the entries in \(Q+\) are only multiplied by 0 when computing the product. We can find a suitable \(Q+\) by finding any orthonormal basis for \(\col(A)^{\perp} = \nul(A^{\transpose})\text{.}\)
