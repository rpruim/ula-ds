Item 6.
If \(C\) is not invertible, then neither is \(C^{\transpose}\text{,}\) so both matrices have determinant 0.
If \(C\) is invertible, then we can write \(C\) as a product of elemntary matrices \(C = E_1 E_2 \cdots E_k\text{.}\) From this we see that
\begin{equation*} \det(C^\transpose) = \det(E_k^\transpose \cdots E_1^\transpose) = \det(E_k^\transpose) \cdots \det(E_1^\transpose)\text{.} \end{equation*}
All tha remains is to show that \(\det(E^\transpose) = \det(E)\) for each elemntary matrix. Each of these is easily verified.
