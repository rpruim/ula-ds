Definition 6.5.11. Coefficient of determination.

For any model that determines predictions \(\yhat\) for a response variable \(\yvec\text{,}\) we can define the coefficient of determination as
\begin{align*} R^2 \amp = \frac{ \len{\yhat - \ybar}^2}{\len{\yvec - \ybar}^2}\text{.} \end{align*}
For linear model with an intercept, we additionally have \(\yvec - \yhat \perp \yhat - \ybar\text{,}\) so
\begin{align*} R^2 \amp = \frac{ \len{\yhat - \ybar}^2}{\len{\yvec - \ybar}^2}\\ \amp = \frac{ \len{\yhat - \ybar}^2}{\len{\yvec - \ybar}^2} \frac{ \len{\yhat - \ybar}^2}{\len{\yvec - \yhat}^2 + \len{\yhat - \ybar}^2 }\text{.} \end{align*}
