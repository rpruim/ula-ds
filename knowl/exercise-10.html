Exercise 4.

This exercise concerns systems of equations having three unknowns. In this case, we know that the solution space of a single equation is a plane. If it helps with visualization, consider using \(3\by5\) index cards to represent planes.
  1. Is it possible that the solution space of a system of four equations in three unknowns is a line? Either sketch an example or give a reason why it can’t happen.
  2. What would you usually expect for the solution space to a system of four equations in three unknowns?
  3. Is it possible that there are no solutions to a system of two equations in three unknowns? Either sketch an example or give a reason why it can’t happen.
  4. Is it possible that there is exactly one solution to a system of two equations in three unknowns? Either sketch an example or give a reason why it can’t happen.
