Interchanges are performed by matrices \(P\) which result from interchanging two rows of the identity matrix. For example,
\begin{equation*} P = \left[\begin{array}{rrr} 0 \amp 1 \amp 0 \\ 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \\ \end{array}\right], \end{equation*}
interchanges the first and second rows of \(I_3\text{.}\) Note that for any such matrix \(P\text{,}\) \(P P = I\text{,}\) so \(\det(P) \det(P) = 1\text{.}\) This means that \(\det(P)\) must either be \(1\) or \(-1\text{.}\) As a transformation, an interchange "swaps" two axes. This reverses the orientation, so
\begin{equation*} \det(P) = -1 \end{equation*}
for any interchange matrix \(P\text{.}\)
