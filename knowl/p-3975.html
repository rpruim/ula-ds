We can solve \(A \xvec = \bvec\) using this factorization as follows:
  • Rewrite as \(PA\xvec = LU \xvec = P\bvec\text{.}\)
  • Solve \(L \cvec = P \bvec\) for \(\cvec\) using backsubstitution.
  • Solve \(U \xvec = \cvec\) for \(\xvec\) backsubstitution.
  • This implies that \(P\bvec = L\cvec = LU \xvec = PA \xvec \text{,}\) so \(A\xvec = \bvec\) since \(P\) is invertible.
