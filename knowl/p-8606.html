  1. The number of columns of \(A^+\) is the number of columns of \(U_r^{\transpose}\text{,}\) which is \(m\text{,}\) and the number of rows is the number of rows of \(V_r\text{,}\) which is \(n\text{.}\)
  2. If \(A\) is invertible, then \(A\) is an \(m\by m\) matrix whose rank is \(m\text{.}\) The reduced singular value decomposition is therefore \(A=U\Sigma V^{\transpose}\) so that \(A^+=V\Sigma^{-1}U^{\transpose} = (U\Sigma V^{\transpose})^{-1}=A^{-1}\text{.}\)
  3. \(AA^{+} = (U_r\Sigma_rV_r^{\transpose})(V_r\Sigma^{-1}U_r^{\transpose}) = U_rU_r^{\transpose}\text{,}\) which projects vectors orthogonal onto \(\col(A)\) since the columns of \(U_r\) are an orthonormal basis for \(\col(A)\text{.}\)
  4. \(AA^{+} = (V_r\Sigma^{-1}U_r^{\transpose})(U_r\Sigma_rV_r^{\transpose}) V_rV_r^{\transpose}\text{,}\) which projects vectors orthogonal onto \(\col(A^{\transpose})\) since the columns of \(V_r\) are an orthonormal basis for \(\col(A^{\transpose})\text{.}\)
