As seen in this activity, row operations can be used to compute the determinant of a matrix. More specifically, applying the forward substitution phase of Gaussian elimination to the matrix \(A\) requires only replacement operations and interchanges and leads us to an upper triangular matrix \(U\) so that \(A\sim U\) and \(\det(A) = \pm \det(U)\text{.}\) And \(\det(U)\) is easily calculated using PropositionÂ 4.5.15. So we see that we can calculate determinates without completing the RREF algorithm. If suffices to do the forward substitutions phase.
