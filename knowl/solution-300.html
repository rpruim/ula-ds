Solution 6.1.5.10.1.

Suppose \(\xvec = a_1\vvec_1 + a_2 \vvec_2 + \cdots + \a_k \vvec_k = \zerovec\) for some scalars \(a_1, a_2, \dots, a_k\text{.}\) This implies that \(\xvec \cdot \vvec_i = \xvec \cdot a_i \vvec_i = 0\text{,}\) so \(a_i = 0\) for all \(i\text{.}\) Since the only linear combination that is the zero vector uses all zero weights, the vectors are linearly independent.
