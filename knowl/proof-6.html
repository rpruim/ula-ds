Proof

If \(A\) is an \(n \by n\) triangular matrix, then we can reduce \(A\) to an \(n \by n\) diagonal matrix \(D\) using only replacement operations. Thus \(\det(A) = \det(D)\text{.}\) Also, the diagonal entries of \(D\) are the same as the diagonal entries of \(A\text{.}\)
A square diagonal matrix \(D\) can be reduced to \(I\) by a sequence of scaling operations. If the diagonal entires of \(D\) are \(d_1, d_2, \dots d_n\text{,}\) then we must scale by \(1/d_1, 1/d_2, \dots 1/d_n\text{,}\) so
\begin{equation*} \frac{1}{d_1} \cdot \frac{1}{d_2} \dots \frac{1}{d_n} \det(D) = \det(I) = 1 \end{equation*}
and
\begin{equation*} \det(D) = d_1 d_2 \cdots d_n\text{.} \end{equation*}
