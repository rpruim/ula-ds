Example 2.4.2. Finding pivots in Python.

When we compute the reduced row echelon form of a matrix using sympy.rref(), the second part of the returned result tells us where the pivots positions are.
Executing this code shows us that the pivot positions (in Python 0-based indexing) are \((0,0), (1,1), (2,3))\text{.}\) Mathematically, using 1-based indexing, we would call these \((1,1), (2,2), (3,4))\text{.}\)
