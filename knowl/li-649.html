Item 2.3.5.8.c.
Notice that \(A(\xvec_h + \xvec_p) = A\xvec_h + A\xvec_p = \zerovec + \bvec = \bvec\text{.}\) This shows that if we add a vector from the solution space to \(A\xvec=\zerovec\) to a vector from the solution space to \(A\xvec=\bvec\text{,}\) we obtain a vector in the solution space to \(A\xvec=\bvec\text{.}\)
