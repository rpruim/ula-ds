While every vector in the domain of a matrix transformation can be provided as in input, it may or may not be the case that every vector in the codomain is a possible output of the transformation. We will call the set of vectors \(\bvec\) that are posible outputs (that is, for which there is a vector \(\xvec \) in the domain with \(A \xvec = \bvec\)) the range of the matrix transformation. . If the range is equal to all of the codomain, then we say that the matrix transformation is onto. This is the same as saying that \(A \xvec = \bvec\) has a solution \(\xvec\) for any \(\bvec\) in the codomain.
