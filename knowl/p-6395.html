This activity introduces an important way of modifying an orthogonal set so that the vectors in the set have unit length. Since \(\len{s \wvec} = s \len{\wvec}\) for any vector \(\wvec\) and scalar \(s\text{,}\)
\begin{equation*} \len{\frac{\wvec}{\len{\wvec}}} = \frac{\len{\wvec}}{\len{\wvec}} = 1\text{.} \end{equation*}
That is, if we divide each vector in an orthogonal set by its length, the result is an orthogonal set of vectors each of which is a unit vector.
