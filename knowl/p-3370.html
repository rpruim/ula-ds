Let \(A = \left[\begin{array}{rr} a \amp b \\ c \amp d \\ \end{array}\right] \text{.}\) We can create an equivalent triangular matrix using \(-\frac{c}{a} R_1 + R_2 \to R_2\text{.}\) This gives us
\begin{equation*} \det A = \det \left[\begin{array}{rr} a \amp b \\ c \amp d \\ \end{array}\right] = \det \left[\begin{array}{rr} a \amp b \\ 0 \amp -\frac{c}{a} b + d \\ \end{array}\right] = a (-\frac{c}{a} b + d) = ad-bc\text{.} \end{equation*}
