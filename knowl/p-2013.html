If we scale a vector and then rotate, we get the same vector as when we rotate first and then scale, so the first linearity property is satisfied. Similarly, for addition, we can either rotate both vectors and then add or add the two vectors and then rotate. We get the same result either way. Since rotations satisfy the two linearity properties, they are linear transforamtions, and therefore matrix transformations. The matrix is completely determined by what the rotation does to \(\evec_1\) and \(\evec_2\text{.}\)
