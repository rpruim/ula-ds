Here are some properties of the matrix transpose, expressed in terms of general matrices \(A\text{,}\) \(B\text{,}\) and \(C\text{.}\) We assume that \(C\) is a square matrix.
  • If \(A+B\) is defined, then \((A+B)^{\transpose} = A^{\transpose}+B^{\transpose}\text{.}\)
  • \((sA)^{\transpose} = sA^{\transpose}\text{.}\)
  • \((A^{\transpose})^{\transpose} = A\text{.}\)
  • \(\det(C) = \det(C^{\transpose})\text{.}\)
  • If \(AB\) is defined, then \((AB)^{\transpose} = B^{\transpose}A^{\transpose}\text{.}\) Notice that the order of the multiplication is reversed.
  • If \(C\) is invertible, then so is \(C^{\transpose}\text{,}\) and \((C^{\transpose})^{-1} = (C^{-1})^{\transpose}\text{.}\)
