Let \(T\) be the matrix transforamtion associated with matrix \(A\text{,}\) let \(c\) be a scalar, and let \(\xvec\) and \(\yvec\) be a vector in the domain of \(T\text{.}\) Then
  1. \(\displaystyle T(c \xvec) = c T(\xvec)\)
  2. \(\displaystyle T(\xvec + \yvec) = T(\xvec) + T(\yvec)\)
These two properties can be combined into a single property if we like:
\begin{equation*} T(c_1 \xvec + c_2 \yvec) = c_1 T(\xvec) + c_2 T(\yvec)\text{.} \end{equation*}
