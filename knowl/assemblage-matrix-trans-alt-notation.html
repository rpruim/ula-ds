Alternative notation.

We have been writing vectors as, for example, \(\threevec 123\text{.}\) To save space vertically, sometimes this is written as \(\langle 1, 2, 3 \rangle\) instead.
Because matrix transformations are so closely connected to their defining matrices, some authors will write the matrix transformation associated with matrix \(A\) as \(A()\) rather than introducing a new letter for the transformation. The round parentheses indicate that we are thinking of \(A\) as a function or transformation. In this notation, \(A(\xvec) = A \xvec\text{.}\)
Combining the two shorthands, we can write things like \(A(\langle 1, 2 ,3 \rangle) = \langle 4, 5 \rangle\) instead of \(T\left(\threevec 123\right) = \twovec 45\) where \(T\) is the matrix transformation associated with the matrix \(A\text{.}\)
