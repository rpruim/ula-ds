  1. Show that if a matrix \(A\) has left inverse \(L\) and right inverse \(R \text{,}\) then \(L = R \text{.}\) Hint: Consider the product \(LAR\text{.}\)
  2. Show that if an \(m \by n\) matrix \(A\) has right inverse \(R\text{,}\) then \(n \ge m\)
  3. Show that if an \(m \by n\) matrix \(A\) has left inverse \(L\text{,}\) then \(m \ge n\text{.}\) Hint: What does \(LA\xvec = L\bvec\) tell you about the equation \(A \xvec = \bvec\text{?}\)
  4. Show that if an \(m \by n\) matrix \(A\) has both a left and a right inverse, then \(m = n\text{,}\) so \(A\) is a square matrix.
