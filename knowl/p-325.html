The solution space to the equation \(A\xvec = \bvec\) is the same as the solution space to the linear system using coefficients from \(A\) and \(b\text{.}\) We can represent all these coefficients in a single autmented matrix: \(\left[ \begin{array}{r|r} A \amp \bvec \end{array}\right]\text{.}\)
