1. Because \(B^{\transpose} = (A^{\transpose}A)^{\transpose} = A^{\transpose}A = B\text{,}\) the matrix \(B\) is symmetric and hence orthogonally diagonalizable.
  2. \(\vvec\cdot(B\vvec) = \vvec\cdot(A^{\transpose}A\vvec) = (A\vvec)\cdot(A\vvec) = |A\vvec|^2\text{.}\)
  3. \(\displaystyle |A\uvec|^2 = \uvec\cdot(B\uvec) = \lambda\uvec\cdot\uvec = \lambda\)
  4. Because \(\lambda=|A\uvec|^2 \geq 0\text{.}\)
  5. In the same way, \(C=\frac1N AA^{\transpose}\) so that \(\uvec\cdot(C\uvec) = \frac1N(A^{\transpose}\uvec)\cdot(A^{\transpose}\uvec) = \frac1N |A^{\transpose}\uvec|^2\text{.}\) Therefore, \(\lambda = \frac1N|A^{\transpose}\uvec|^2\text{.}\)
