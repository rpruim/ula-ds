Item 5.3.5.3.b.
True. If \(A\) and \(B\) are similar, then \(A=PBP^{-1}\) so that \(\det A = \det B\text{.}\) If \(A\) is invertible, we know that \(\det A \neq 0\text{,}\) which also tells us that \(\det B\neq 0\) and hence that \(B\) is invertible.
We can determine the inverse of \(B\) as follows. \(P^{-1}AP = B\) and \(P^{-1}AP P^{-1} A^{-1} P = I\text{,}\) so \(B^{-1} = P^{-1} A^{-1} P\text{.}\)
