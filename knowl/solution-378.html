Solution 7.1.4.10.1.

  1. If \(Q\) is a \(3\by2\) matrix whose columns are an orthonormal basis for \(W\text{,}\) then \(P=QQ^{\transpose}\text{.}\) This means that \(P^{\transpose}=(Q^{\transpose})^{\transpose}Q^{\transpose} = QQ^{\transpose} = P\) so \(P\) is symmetric and hence orthogonally diagonalizable.
  2. If \(\vvec\) is in \(W\text{,}\) then \(P\vvec=\vvec\) and if \(\vvec\) is in \(W^\perp\text{,}\) then \(P\vvec=0\text{.}\) This means that the eigenvalues of \(P\) are either 0 or 1.
  3. The eigenspaces \(E_1=W\) and \(E_0=W^{\perp}\)
